# CSS Basic Project

## Overview
This project is part of the ALX HTML & CSS curriculum.  
It focuses on styling a webpage using **CSS Flexbox** to create a responsive layout with a header, main (article + aside), and footer.

## Features
- Flexbox layout for responsiveness
- Mobile-friendly design with `class="works_on_smartphone"`
- Viewport meta tag for proper scaling
- Custom styles for colors, fonts, and borders
- W3C-compliant HTML and CSS

## Files
- `index.html`
- `tweets.html`
- `base.css`
- `styles.css`
